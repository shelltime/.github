## What is ShellTime?

ShellTime is a developer analytics platform built on three pillars: shell command tracking, AI coding tool usage metrics, and editor activity monitoring. It provides insights into your development workflow across terminals, AI assistants, and IDEs.

## Features

### Command Tracking

- **Shells**: bash, zsh, fish
- Command history sync across machines
- Execution time and exit code analytics
- Weekly usage reports and trends
- Search and filter command history

### AI Coding Analytics

- Track **Claude Code** and **Codex** usage
- Session duration and interaction metrics
- Token usage insights
- Cost estimation and trends

### Editor Activity

- **VS Code**, **JetBrains IDEs**, **Neovim**
- Active coding time tracking
- Project and language breakdown
- Daily and weekly activity reports
